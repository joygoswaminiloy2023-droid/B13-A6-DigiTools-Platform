## Digital Tools Buying Website

### About the Project

This is a simple and user-friendly website where users can explore different digital tools and easily add them to a cart. The goal of this project was to create a clean interface with smooth interactions, just like a real online shopping experience.

### Technologies Used

- React.js
- Tailwind CSS & DaisyUI
- JavaScript (ES6+)
- React-Toastify
- JSON

### Key Features

- **Cart Management:**  
  Users can add products to the cart, remove them anytime, and clear everything with a single checkout button.

- **Easy Navigation:**  
  The interface allows switching between product view and cart view smoothly without reloading the page.

- **Instant Feedback:**  
  Every action like adding or removing a product shows a quick notification, making the experience more interactive.
